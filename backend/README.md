# backend

## Quick start

### Environment

Set your local environment.

```
$ cp .env.default .env
# Edit .env
```

### NVM and Node version

Install [NVM](https://github.com/nvm-sh/nvm) and set the default node version to match:

```
$ nvm -v
0.39.3
$ node -v
v16.14.0
```

### Install dependencies

```
$ npm i
```

### Run backend in dev mode

```
$ npm run dev
```
